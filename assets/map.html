GPX Utility — How to use

Load, inspect, edit and save GPX tracks.

File

Load GPX…Open a GPX file via the left panel. The track is drawn on the map and stats are computed.
Save GPX…Save the modified track (including all edits) to a new GPX file.

Map navigation

Scroll wheelZoom in / out.
DragPan the map.

Point list

Points (N) buttonToggle the floating points panel. Columns: index, time, cumulative distance, gap from previous point, speed, coordinates.

Hover (bidirectional)

Mouse over trackYellow dot appears on the nearest point; corresponding row is highlighted in the list.
Mouse over list rowYellow dot moves to that point on the map.

Selection

Click track or rowSets the anchor point (orange dot, dark border on row). Click again to deselect.
Shift + clickExtends the selection to a range from the anchor to the clicked point. The segment is highlighted in orange on the map.
Map background clickClears the selection.

Context menu (right-click selected segment)

DeleteRemoves the selected points and reconnects the remaining track.
Delete and joinRemoves the selected points and subtracts the deleted segment's time duration from all subsequent point timestamps — closes the time gap as if the segment never happened.
SplitDeletes the selected points and splits the remaining track into two separate segments (blue + purple). Cannot split at the very start or end.
DuplicateInserts a copy of the selected points immediately after the selection, then auto-selects the duplicate.
